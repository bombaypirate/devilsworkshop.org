  A good thing for tweeter fans out there! Came across this unique and interesting thing which I wanted to share with all the readers and specially with those readers who are a big fan of twitter and has a huge number of followings there. This unique service I am talking about is <a href="http://www.tweetbars.com/" onclick="_gaq.push(['_trackEvent', 'outbound-article', 'http://www.tweetbars.com/', 'Tweetbars']);" target="_blank">Tweetbars</a>. Basically, this is a service that creates a block of html codes that you can use as your signature in Emails, Forums, While commenting on blogs, etc. One such signature block is placed here to make you understand better about <a href="http://www.tweetbars.com/" onclick="_gaq.push(['_trackEvent', 'outbound-article', 'http://www.tweetbars.com/', 'tweetbars']);" target="_blank">tweetbars</a>.

So the latest tweet on <a href="http://www.twitter.com/devils_workshop" onclick="_gaq.push(['_trackEvent', 'outbound-article', 'http://www.twitter.com/devils_workshop', '@Devils_Workshop']);" target="_blank">@Devils_Workshop</a> was about the latest post and the same has been displayed above.
