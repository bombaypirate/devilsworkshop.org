We can see many types of tutorial about **Image Reflection in Photoshop**. You can create this effect by using layer mask and gradient effect. In this post we can see how to create Image Reflection in an easier way.

You can create Image Reflection effect in a few simple steps.

I will show steps by which you will be able to take the image to the left and transform it with a reflection like the one to the right. (See image below).

<a rel="attachment wp-att-29879" href="http://devilsworkshop.org/how-to-create-image-reflection-in-photoshop/img-001/"><img class="alignnone size-medium wp-image-29879" title="img-001" src="http://cdn.devilsworkshop.org/files/2010/09/img-001-600x326.jpg" alt="" width="600" height="326" /></a>

Now here below I have included the original Image I will use for this tutorial.

<a rel="attachment wp-att-29878" href="http://devilsworkshop.org/how-to-create-image-reflection-in-photoshop/img-01/"><img class="alignnone size-full wp-image-29878" title="img-01" src="http://cdn.devilsworkshop.org/files/2010/09/img-01.jpg" alt="" width="522" height="363" /></a>

  * Before we start please create duplicate the image.
  * Select the image layer and simply right-click on it for duplicate layer.

When we right-click on the image layer we can see this palate. (See the image below)

<a rel="attachment wp-att-29883" href="http://devilsworkshop.org/how-to-create-image-reflection-in-photoshop/img-02/"><img class="alignnone size-medium wp-image-29883" title="img-02" src="http://cdn.devilsworkshop.org/files/2010/09/img-02-600x438.jpg" alt="" width="600" height="438" /></a>

  * Click on it. We can also give name to this layer.
  * We can also use short cut key** Ctrl+J** to duplicate the image.

Now select the duplicate image layer and place it at below the Original image.

<a rel="attachment wp-att-29885" href="http://devilsworkshop.org/how-to-create-image-reflection-in-photoshop/img-03-2/"><img class="alignnone size-medium wp-image-29885" title="img-03" src="http://cdn.devilsworkshop.org/files/2010/09/img-031-460x600.jpg" alt="" width="460" height="600" /></a>

  * Now we need to reverse the image. Select the image and go to **Edit > Transform > Flip Vertical**.

<a rel="attachment wp-att-29887" href="http://devilsworkshop.org/how-to-create-image-reflection-in-photoshop/img-04-2/"><img class="alignnone size-medium wp-image-29887" title="img-04" src="http://cdn.devilsworkshop.org/files/2010/09/img-041-600x524.jpg" alt="" width="600" height="524" /></a>

Now we have got our reverse image. <img src="http://devilsworkshop.org/wp-includes/images/smilies/simple-smile.png" alt=":-)" class="wp-smiley" style="height: 1em; max-height: 1em;" />

<a rel="attachment wp-att-29888" href="http://devilsworkshop.org/how-to-create-image-reflection-in-photoshop/img-05/"><img class="alignnone size-medium wp-image-29888" title="img-05" src="http://cdn.devilsworkshop.org/files/2010/09/img-05-460x600.jpg" alt="" width="460" height="600" /></a>

Click on the below image and go to Tool bar. Now just right-click on **Marquee Tool**. We will see this pallet

<a rel="attachment wp-att-29889" href="http://devilsworkshop.org/how-to-create-image-reflection-in-photoshop/img-06/"><img class="alignnone size-medium wp-image-29889" title="img-06" src="http://cdn.devilsworkshop.org/files/2010/09/img-06-600x360.jpg" alt="" width="600" height="360" /></a>

Select the **Rectangular Marquee Tool (M)** and make a selection on the below image. Ideally it should be from bottom up to half-size of image like the one shown below.

<a rel="attachment wp-att-29890" href="http://devilsworkshop.org/how-to-create-image-reflection-in-photoshop/img-07/"><img class="alignnone size-medium wp-image-29890" title="img-07" src="http://cdn.devilsworkshop.org/files/2010/09/img-07-600x392.jpg" alt="" width="600" height="392" /></a>

Now keep choice on image and go to **Select > Modify > Feather** we can see this palate.

<a rel="attachment wp-att-29891" href="http://devilsworkshop.org/how-to-create-image-reflection-in-photoshop/img-11/"><img class="alignnone size-full wp-image-29891" title="img-11" src="http://cdn.devilsworkshop.org/files/2010/09/img-11.jpg" alt="" width="565" height="376" /></a>

  * Select the Feather Radius value. Here I take **Feather Radius value -50**. Now just press delete 2-3 time as per your image requirements.

<a rel="attachment wp-att-29892" href="http://devilsworkshop.org/how-to-create-image-reflection-in-photoshop/img-08/"><img class="alignnone size-medium wp-image-29892" title="img-08" src="http://cdn.devilsworkshop.org/files/2010/09/img-08-460x600.jpg" alt="" width="460" height="600" /></a>

Now for make more realistic effect go to layer pallet select Opacity give **Opacity value â€“ 50%.** You can adjust it as per your image requirement.

<a rel="attachment wp-att-29893" href="http://devilsworkshop.org/how-to-create-image-reflection-in-photoshop/img-09/"><img class="alignnone size-medium wp-image-29893" title="img-09" src="http://cdn.devilsworkshop.org/files/2010/09/img-09-600x222.jpg" alt="" width="600" height="222" /></a>

There we go. We have our Image Reflection effect.
