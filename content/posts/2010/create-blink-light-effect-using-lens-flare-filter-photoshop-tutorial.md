You must have seen many animated .GIF images but had not idea on how to go about creating them. Here I am going to tell step by step how to go about creating an image with simple GIF animation.

In this tutorial we are going to use **Blink Light Effect** using **Lens Flare Filter**. As in the above image you can see it the original on the left hand side and make it look like the headlights were turned on.

## Original Image

<a rel="attachment wp-att-31255" href="http://devilsworkshop.org/create-blink-light-effect-using-lens-flare-filter-photoshop-tutorial/image-2-6/"><img class="alignnone size-full wp-image-31255" title="image-2" src="http://cdn.devilsworkshop.org/files/2010/10/image-22.jpg" alt="" width="500" height="375" /></a>

  * Lets first make a duplicate layer of the image. **Right Click** on the layer and you can see the **Duplicate Layer** option in the below palate. Or just PressÂ  **Ctrl+J** key. See the image below.

<a rel="attachment wp-att-31256" href="http://devilsworkshop.org/create-blink-light-effect-using-lens-flare-filter-photoshop-tutorial/image-3-6/"><img class="alignnone size-full wp-image-31256" title="image-3" src="http://cdn.devilsworkshop.org/files/2010/10/image-32.jpg" alt="" width="392" height="222" /></a>

  * After duplicating the layer just go to **Filter > Render > Lens Flare**

<a rel="attachment wp-att-31257" href="http://devilsworkshop.org/create-blink-light-effect-using-lens-flare-filter-photoshop-tutorial/image-4-5/"><img class="alignnone size-full wp-image-31257" title="image-4" src="http://cdn.devilsworkshop.org/files/2010/10/image-42.jpg" alt="" width="356" height="354" /></a>

  * Now we can see the Lens Flare palate box displayed on your screen.
  * Set the Lens Flare on the car&#8217;s headlight.
  * Set the **Brightness value &#8211; 85%**, keep on **105mm Prime**. Click on &#8220;OK&#8221;.

<a rel="attachment wp-att-31258" href="http://devilsworkshop.org/create-blink-light-effect-using-lens-flare-filter-photoshop-tutorial/image-5-5/"><img class="alignnone size-full wp-image-31258" title="image-5" src="http://cdn.devilsworkshop.org/files/2010/10/image-52.jpg" alt="" width="375" height="388" /></a>

  * Now repeat this step with the left side headlight.
  * Now we have created a light effect by using lends flare filter. Check the image below.

<a rel="attachment wp-att-31260" href="http://devilsworkshop.org/create-blink-light-effect-using-lens-flare-filter-photoshop-tutorial/image-7-5/"><img class="alignnone size-full wp-image-31260" title="image-7" src="http://cdn.devilsworkshop.org/files/2010/10/image-72.jpg" alt="" width="500" height="375" /></a>

  * Now to add animation effect to it.
  * We have two layers. One is normal image and another is with the light effect.
  * Just go to **Window > Animation**. You can see **Animation Frames Pallet** display on your screen. (Check image below)

<a rel="attachment wp-att-31261" href="http://devilsworkshop.org/create-blink-light-effect-using-lens-flare-filter-photoshop-tutorial/image-8-5/"><img class="alignnone size-full wp-image-31261" title="image-8" src="http://cdn.devilsworkshop.org/files/2010/10/image-82.jpg" alt="" width="482" height="414" /></a>

  * Here we see the frame 1 in the **Animation Frames** Pallet.

<a rel="attachment wp-att-31262" href="http://devilsworkshop.org/create-blink-light-effect-using-lens-flare-filter-photoshop-tutorial/image-9-4/"><img class="alignnone size-full wp-image-31262" title="image-9" src="http://cdn.devilsworkshop.org/files/2010/10/image-91.jpg" alt="" width="380" height="226" /></a>

  * Now to create the duplicate of frame 1.
  * Just Click on this icon to **Duplicate the frame**.

<a rel="attachment wp-att-31263" href="http://devilsworkshop.org/create-blink-light-effect-using-lens-flare-filter-photoshop-tutorial/image-10-4/"><img class="alignnone size-full wp-image-31263" title="image-10" src="http://cdn.devilsworkshop.org/files/2010/10/image-101.jpg" alt="" width="333" height="195" /></a>

  * Select the first frame and go to the **Layer Pallet** and then hide the layer. See the red highlighted box in the image

<a rel="attachment wp-att-31264" href="http://devilsworkshop.org/create-blink-light-effect-using-lens-flare-filter-photoshop-tutorial/image-11-4/"><img class="alignnone size-full wp-image-31264" title="image-11" src="http://cdn.devilsworkshop.org/files/2010/10/image-111.jpg" alt="" width="575" height="315" /></a>

  * Now select the second frame and go to **Layer Pallet**
  * Keep the selection on the above image layer. (Check the image below)

<a rel="attachment wp-att-31265" href="http://devilsworkshop.org/create-blink-light-effect-using-lens-flare-filter-photoshop-tutorial/image-12-3/"><img class="alignnone size-medium wp-image-31265" title="image-12" src="http://cdn.devilsworkshop.org/files/2010/10/image-121-600x289.jpg" alt="" width="600" height="289" /></a>

  * We have two frames and now to set the time to display the frame.
  * Set the **Time Value &#8211; 0.5sec**. for both images. (Check the image below)

<a rel="attachment wp-att-31266" href="http://devilsworkshop.org/create-blink-light-effect-using-lens-flare-filter-photoshop-tutorial/image-13-3/"><img class="alignnone size-full wp-image-31266" title="image-13" src="http://cdn.devilsworkshop.org/files/2010/10/image-131.jpg" alt="" width="302" height="265" /></a>

  * We can preview the Animation here. Just click the **Play Button** below the Frames Pallet.

<a rel="attachment wp-att-31267" href="http://devilsworkshop.org/create-blink-light-effect-using-lens-flare-filter-photoshop-tutorial/image-14-2/"><img class="alignnone size-full wp-image-31267" title="image-14" src="http://cdn.devilsworkshop.org/files/2010/10/image-141.jpg" alt="" width="248" height="183" /></a>

  * Save the file in GIF Animated format.
  * Go to **File > Save for Web & Devices (Alt+Shift_Ctrl+S)**. Click on it.

<a rel="attachment wp-att-31268" href="http://devilsworkshop.org/create-blink-light-effect-using-lens-flare-filter-photoshop-tutorial/image-15/"><img class="alignnone size-full wp-image-31268" title="image-15" src="http://cdn.devilsworkshop.org/files/2010/10/image-151.jpg" alt="" width="340" height="349" /></a>

  * **Save for Web & Devices** is displayed on the screen.

<a rel="attachment wp-att-31269" href="http://devilsworkshop.org/create-blink-light-effect-using-lens-flare-filter-photoshop-tutorial/image-16/"><img class="alignnone size-medium wp-image-31269" title="image-16" src="http://cdn.devilsworkshop.org/files/2010/10/image-161-600x448.jpg" alt="" width="600" height="448" /></a>

  * Now select the GIF format here like shown in the image below. Select **GIF** and Save the file.
