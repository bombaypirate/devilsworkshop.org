In this post we are going to see how to create a Grunge Border Effect to Solid Color Box with using filter. We can see so many of Border Styles in **Photoshop**. I thought I will write on how to create Grunge Border Effect in a few simple steps.

<a rel="attachment wp-att-30319" href="http://devilsworkshop.org/how-to-create-the-grunge-border-effect-to-solid-color-box-with-photoshop/images1-3/"><img class="alignnone size-medium wp-image-30319" title="images1" src="http://cdn.devilsworkshop.org/files/2010/09/images1-600x205.jpg" alt="" width="600" height="205" /></a>

First create a new file in Photoshop. Give background color like below image.

<a rel="attachment wp-att-30320" href="http://devilsworkshop.org/how-to-create-the-grunge-border-effect-to-solid-color-box-with-photoshop/images2-2/"><img class="alignnone size-full wp-image-30320" title="images2" src="http://cdn.devilsworkshop.org/files/2010/09/images2.jpg" alt="" width="500" height="342" /></a>

Then draw the box on the new layer. Go to Toolbar and select the **Rectangle Tool (U)**

<a rel="attachment wp-att-30321" href="http://devilsworkshop.org/how-to-create-the-grunge-border-effect-to-solid-color-box-with-photoshop/images3/"><img class="alignnone size-full wp-image-30321" title="images3" src="http://cdn.devilsworkshop.org/files/2010/09/images3.jpg" alt="" width="303" height="276" /></a>

This way we have finished creating a box. Here I given it a Lemon Yellow color. (see image below)

<a rel="attachment wp-att-30322" href="http://devilsworkshop.org/how-to-create-the-grunge-border-effect-to-solid-color-box-with-photoshop/images4/"><img class="alignnone size-full wp-image-30322" title="images4" src="http://cdn.devilsworkshop.org/files/2010/09/images4.jpg" alt="" width="500" height="342" /></a>

After that give it stroke of a dark shade of yellow. Just go to **Layer Pallet >Layer Style > Stroke. **

<a rel="attachment wp-att-30323" href="http://devilsworkshop.org/how-to-create-the-grunge-border-effect-to-solid-color-box-with-photoshop/images5/"><img class="alignnone size-full wp-image-30323" title="images5" src="http://cdn.devilsworkshop.org/files/2010/09/images5.jpg" alt="" width="419" height="333" /></a>

As you click on stroke you can see Layer Style pallet on your screen. Now keep your Stroke setting as shown in image below. Keep stroke **size-1px, Inside**.

<a rel="attachment wp-att-30324" href="http://devilsworkshop.org/how-to-create-the-grunge-border-effect-to-solid-color-box-with-photoshop/images6/"><img class="alignnone size-medium wp-image-30324" title="images6" src="http://cdn.devilsworkshop.org/files/2010/09/images6-600x466.jpg" alt="" width="600" height="466" /></a>

You can see the solid color border to the box.

<a rel="attachment wp-att-30325" href="http://devilsworkshop.org/how-to-create-the-grunge-border-effect-to-solid-color-box-with-photoshop/images6-1/"><img class="alignnone size-full wp-image-30325" title="images6-1" src="http://cdn.devilsworkshop.org/files/2010/09/images6-1.jpg" alt="" width="502" height="342" /></a>

Now let&#8217;s see how to convert it into **Grunge Border Effect**.

  * Just go to **Filter >> Stylize >> Diffuse**. Click **OK**.
  * Now you can see the Diffuse box displayed on your screen. Just select the **Normal** option and click Ok.

<a rel="attachment wp-att-30326" href="http://devilsworkshop.org/how-to-create-the-grunge-border-effect-to-solid-color-box-with-photoshop/images7/"><img class="alignnone size-full wp-image-30326" title="images7" src="http://cdn.devilsworkshop.org/files/2010/09/images7.jpg" alt="" width="478" height="417" /></a>

Here we get the Grunge Border Effect Box. We create this effect by using** **the **Diffuse Filter**.

<a rel="attachment wp-att-30327" href="http://devilsworkshop.org/how-to-create-the-grunge-border-effect-to-solid-color-box-with-photoshop/images8/"><img class="alignnone size-full wp-image-30327" title="images8" src="http://cdn.devilsworkshop.org/files/2010/09/images8.jpg" alt="" width="500" height="342" /></a>

Now for some more finishing touches.

I applied **Outer Glow** to the image. Go to Layer Style and select the Outer glow option and keep your setting similar to the image below.

<a rel="attachment wp-att-30328" href="http://devilsworkshop.org/how-to-create-the-grunge-border-effect-to-solid-color-box-with-photoshop/images9/"><img class="alignnone size-medium wp-image-30328" title="images9" src="http://cdn.devilsworkshop.org/files/2010/09/images9-600x413.jpg" alt="" width="600" height="413" /></a>

Now we have the Grunge Border Effect to a Solid Color Box with Photoshop.

<a rel="attachment wp-att-30329" href="http://devilsworkshop.org/how-to-create-the-grunge-border-effect-to-solid-color-box-with-photoshop/images10/"><img class="alignnone size-full wp-image-30329" title="images10" src="http://cdn.devilsworkshop.org/files/2010/09/images10.jpg" alt="" width="500" height="342" /></a>

In this way you can give a Grunge Border effect to the Solid Box within a couple of minutes. Do you have any alternative steps or methods with Photoshop? Do drop in your comments.
