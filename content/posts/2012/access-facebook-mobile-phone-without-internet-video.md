Facebook India recently <a href="https://www.facebook.com/FacebookIndia/posts/344648582213400" onclick="_gaq.push(['_trackEvent', 'outbound-article', 'https://www.facebook.com/FacebookIndia/posts/344648582213400', 'announced']);" >announced</a> that mobile users in India can also access their Facebook accounts without the need of a internet connection, with the help of an app called Fonetwish.

All you are need to do is dial ***325#** on your mobile phone. This will pull up a list of options via a console like interface to access your Facebook news feeds, update your status, manage friend requests and notifications.

### Using Fonetwish App for Facebook on your Mobile Phone

  * This is done via using Facebook App called <a href="https://apps.facebook.com/fonetwish/" onclick="_gaq.push(['_trackEvent', 'outbound-article', 'https://apps.facebook.com/fonetwish/', 'Fonetwish']);" >Fonetwish</a>, which charges Re 1/- per day while updating status on Facebook is free.
  * This service is meant to work on any phone which could be an iPhone or even simpler lower-end phone models. There are no roaming charges either.
  * The only problem with using Fonetwish is that it has a limit of 140 characters.
  * This service is available in India on Airtel, Aircel, Idea, Videocon, Loop Mobile and Tata Docomo networks.
