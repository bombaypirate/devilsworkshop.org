I love to write without any distractions and it is one of the reasons, I like using Windows Live Writer for blogging. I like to write parts of a post first and then go about adding images or videos into them.

If you are using Chrome, then you might like *<a href="https://chrome.google.com/webstore/detail/pnengefjfhgcceajaepbjhanoojifmog" onclick="_gaq.push(['_trackEvent', 'outbound-article', 'https://chrome.google.com/webstore/detail/pnengefjfhgcceajaepbjhanoojifmog', 'Writer Web App']);" >Writer Web App</a>* which allows users to quickly type in a distraction free environment.

<a href="http://devilsworkshop.org/distraction-free-writer-chrome-browser/chrome_distraction_free_writer/" rel="attachment wp-att-57430"><img class="alignnone size-full wp-image-57430" style="border-image: initial; border-width: 1px; border-color: black; border-style: solid;" title="Distraction Free Writer for Chrome" src="http://cdn.devilsworkshop.org/files/2012/05/Chrome_Distraction_free_Writer.png" alt="" width="669" height="518" /></a>

The web app calls itself the typewriter for the internet. It expectantly has a sparse design and allows users to save notes. It also allows sending what you have written on various blogging platforms like Blogger, WordPress, LiveJournal and more. Users can even send an email from the web app.

The online tool is quite perfect if you want to quickly write down some thoughts without any distraction. In case you want a software that is distraction free on your desktop, you might like trying out [CreaWriter for Windows][1] or [OmmWriter for Mac and iPad][2]. In case you are using [Linux using JDarkroom][3] is quite useful.

Try out ***Writer for Chrome*** and drop in your comments.
