They are freely available on the Android Marketplace for Apps. It&#8217;s easy to download and install any app via Android Market.

Android allows apps to be installed via marketplace only due to security issues and malware problems. But there are many software that are not available in the market, several customized applications, apps created by your friends etc.

You may face problems installing them. Here, in this simple tutorial, I will guide you how to install any apps from any site to your Android phone.

## Follow me

  * Download/Copy  to your SD Card,  the app installer from the site you want (it must have a &#8220;**.apk**&#8221; extension )
  * Now Navigate to Home -> Settings.

<p style="text-align: center;">
  <a rel="attachment wp-att-40466" href="http://devilsworkshop.org/install-apps-site-android-market/instll-apps1/"><img class="aligncenter size-full wp-image-40466" title="instll apps1" src="http://cdn.devilsworkshop.org/files/2011/05/instll-apps1.png" alt="" width="320" height="480" /></a>
</p>

  * In the settings menu, select Applications

<a rel="attachment wp-att-40467" href="http://devilsworkshop.org/install-apps-site-android-market/instll-apps2/"><img class="aligncenter size-full wp-image-40467" title="instll apps2" src="http://cdn.devilsworkshop.org/files/2011/05/instll-apps2.png" alt="" width="320" height="480" /></a>

  * In the application menu, you can see the option named &#8216;Unknown Sources&#8217;. Tap on it.

<a rel="attachment wp-att-40468" href="http://devilsworkshop.org/install-apps-site-android-market/instll-apps3/"><img class="aligncenter size-full wp-image-40468" title="instll apps3" src="http://cdn.devilsworkshop.org/files/2011/05/instll-apps3.png" alt="" width="320" height="480" /></a>

In the box that appear, Select **&#8220;OK&#8221;**.<a rel="attachment wp-att-40469" href="http://devilsworkshop.org/install-apps-site-android-market/instll-apps4/"><img class="aligncenter size-full wp-image-40469" title="instll apps4" src="http://cdn.devilsworkshop.org/files/2011/05/instll-apps4.png" alt="" width="320" height="480" /></a>

  * Now select the app .apk file and tap on it

<p style="text-align: center;">
  <a rel="attachment wp-att-40470" href="http://devilsworkshop.org/install-apps-site-android-market/device1/"><img class="aligncenter" title="device1" src="http://cdn.devilsworkshop.org/files/2011/05/device1.png" alt="" width="320" height="480" /></a>
</p>

  * Select install and it will be done!

<a rel="attachment wp-att-40471" href="http://devilsworkshop.org/install-apps-site-android-market/device2/"><img class="aligncenter size-full wp-image-40471" title="device2" src="http://cdn.devilsworkshop.org/files/2011/05/device2.png" alt="" width="320" height="480" /></a>

Now the app have been installed on your phone. If you want to install an app from your computer, simply copy it to your SD Card / mail it to your android phone and carry out the same process to install it. <img src="http://devilsworkshop.org/wp-includes/images/smilies/simple-smile.png" alt=":-)" class="wp-smiley" style="height: 1em; max-height: 1em;" />
